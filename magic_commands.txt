finding function actual type (still a little bit obfuscated)
bpftool btf dump file /sys/kernel/btf/vmlinux | grep -A 5 "\[133367\]"

removing scheduler: 
sudo rm /sys/fs/bpf/sched_ext/sched_ops 

compile: 
sudo clang -target bpf -g -O2 -c minimal_sched.bpf.c  -o minimal_sched.bpf.o -I.

run: 
sudo bpftool struct_ops register minimal_sched.bpf.o /sys/fs/bpf/sched_ext

stress tool: 
stress -c 10 

some err output: 
sudo dmesg | tail -n 40

seeing printk logs: 
sudo bpftool prog tracelog
or
sudo cat /sys/kernel/debug/tracing/trace_pipe

installing newest libbpf: 
git clone https://github.com/libbpf/libbpf.git
cd libbpf/src && make
